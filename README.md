# NM i AI 2026

Dette repoet inneholder mitt arbeid for NM i AI 2026.

Konkurransen består av tre uavhengige AI-oppgaver som publiseres ved kickoff. Hver oppgave utvikles i sin egen mappe for å holde kode, eksperimenter og submissions ryddig separert.

## Struktur

- `task1-Tripletex/` – løsning for task 1
- `task2-Astar/` – løsning for task 2
- `task3-Norgesgruppen/` – løsning for task 3

Hver task-mappe vil inneholde:
- kode for dataforberedelse, trening og/eller inferens
- eventuelle notebooks brukt til utforsking
- dokumentasjon av fremgangsmåte
- filer for submission hvis relevant

## Arbeidsmåte

Målet med repoet er å dokumentere en ryddig og etterprøvbar arbeidsprosess under konkurransen. Kode, eksperimenter og leveranser organiseres per oppgave.

## Status

Konkurransen starter torsdag 19.03.2026 kl. 18:00 CET og varer til søndag kl. 15:00 CET. Task-spesifikke regler, datasett og innleveringsformat publiseres ved kickoff.
